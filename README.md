# gta-sa-plants-vs-zombies
## Requirements
1. SA:MP
2. ASI Loader
3. SAMPFUNCS
4. MoonLoader
5. mimgui

## Development
1. Download moonly
2. clone repo into `GTA_SA_PATH\moonly`
3. connect to any SA:MP server and use `/pvz`
   
## Building:
1. 'go run generate-config.go && lubu.exe bundle-config.json'
